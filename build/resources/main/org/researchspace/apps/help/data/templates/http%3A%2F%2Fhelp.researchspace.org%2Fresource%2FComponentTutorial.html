Hello World Component

The goal is to create the most simple "Hello World!"-like component.

First, we need to create the JavaScript implementation for our new component. Put the code below into platform-web/src/main/components/GreetingComponent.tsx file. If this is your first encounter with React.js we recommend first to read React Getting Started. The pretty HTML-like syntax we use is JSX, but it would work equally well without it.

Please, keep in mind that we have to import React to make JSX syntax work. Some outdated JavaScript IDE may still claim that this import is extraneous, but it's not.
