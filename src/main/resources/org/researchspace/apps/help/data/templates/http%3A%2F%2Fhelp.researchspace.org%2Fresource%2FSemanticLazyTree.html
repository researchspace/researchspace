Semantic lazy tree

The component is best for tree-based visualization, navigation, browsing through sub-tree structures or hierarchies with ability to use keyword filtering.

The main difference from semantic tree is that data in Lazy Tree is lazily loaded by small portions as the user explores the hierarchy by expanding branches. This allows the component to render large hierarchical structures with thousands of nodes without any large impact on the performance.
