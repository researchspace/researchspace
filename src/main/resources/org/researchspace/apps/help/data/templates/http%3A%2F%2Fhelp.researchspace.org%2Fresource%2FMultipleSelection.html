Multiple Selection

The platform provides the set of components that allow selecting multiple resources and performing actions with them:

  • - triggers the Components.Selection.Toggle event on resource selection
  • - collects selected resources and displays possible actions
  • - container component that stores selected resources and allows restoring the state of the mp-selection-toggle component if it gets re-rendered

The following attributes can be configured for the component:

selection: string

Name of checkbox listener

tag: string

Extra data to pass to listener, so that it's possible to figure out, which of checkboxes was toggled

The component allows specify the following attributes:

id: string

Required to make dropdown available to screen reader software

selection: string

Action group name

title: string

Dropdown caption

style?: string

CSS style

class?: string

CSS class
