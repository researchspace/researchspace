How to download and install the ResearchSpace platform

Version:   ResearchSpace 4.0.0
Latest review:   9 Jan, 2025

To receive support on ResearchSpace download and installation contact us at support@kartography.org.

Download

ResearchSpace is free to download with open source initiative licence.

The Desktop version of ResearchSpace can be downloaded and installed on Windows, Mac OS or Linux.

Installation

The ResearchSpace platform can be installed on Desktop as an application using Docker or running locally the ZIP file bundle.

Requirements

The platform's browser compatibility is Google Chrome (minimum version 53) and Mozilla Firefox (minimum version 58). Use of ResearchSpace in other browsers or older versions of Chrome or Firefox is not currently supported.

Installation with Docker

To start the ResearchSpace platform:

  • Install Docker desktop at https://www.docker.com/get-started, following the Docker installation guide.
  • Open Docker desktop (no login is needed) and keep it running in background.
  • Download the file ResearchSpace 4.0.0 for Docker.

  • Expand the zip file, open the terminal and move to the downloaded directory with the followig command

    cd yourDirectoryPath/researchspace-docker-desktop-main

    NOTE: in the command replace yourDirectoryPath with your local directory path.

  • Then run the followig command from terminal

    Start the Docker application with command docker compose up -d.

    docker compose up -d

  • The platform might still be loading, so check the platform loading status with command

    docker logs localhost-researchspace-1

    The platform starts when the logs end with the following message

                INFO:oejs.AbstractConnector:main: Started ServerConnector@40fd1a78{HTTP/1.1, (http/1.1)}{0.0.0.0:8080}
    
            INFO:oejs.Server:main: Started

    So if necessary rerun command docker logs localhost-researchspace-1 until the message is displayed.

    The platform might still be loading, so check the platform loading status with command docker logs localhost-researchspace-1.

    The platform starts when the logs end with the following message

    INFO:oejs.AbstractConnector:main: Started ServerConnector@40fd1a78{HTTP/1.1, (http/1.1)}{0.0.0.0:8080} INFO:oejs.Server:main: Started

  • Open http://localhost:10214/login in your browser.

  • Log into the platform with username:admin and password:admin.

  • To shut down the platform run command

    docker compose down
For more details look at ResearchSpace setup for desktop using Docker.

Start ResearchSpace/docker with example data

To explore ResearchSpace starting with an example of research data
follow these steps:
  • Download and expand the zip file example-runtime-data-for-docker.
  • Open the Researchspace folder researchspace-docker-desktop-main previously downloaded for the installation with docker.
  • Inside researchspace-docker-desktop-main:
    • replace the folder researchspace/runtime-data with the folder with the same name in example-runtime-data-for-docker.
    • add the folder blazegraph that you can find in example-runtime-data-for-docker.
  • Finally start the ResearchSpace platform with docker as described above.

Installation with ZIP

Requirements

ResearchSpace is a Java-based application and it requires your system has Java 11 installed.

To verify your system has Java 11 running or install the appropriate version follow the Java 11 installation guide.

If your system has several versions of Java installed, edit all `.sh` or `.bat` files and specify the path to Java 11.

For example in start.sh the line

java -Dfile.encoding=UTF8 -cp "${DIR}/starter/*:${DIR}/starter/logback-config" org.akhikhl.gretty.GrettyStarter $@ start

will be modified to something like

/Library/Java/JavaVirtualMachines/temurin-11.jdk/Contents/Home/bin/java -Dfile.encoding=UTF8 -cp "${DIR}/starter/*:${DIR}/starter/logback-config" org.akhikhl.gretty.GrettyStarter $@ start
where /Library/Java/JavaVirtualMachines/temurin-11.jdk/Contents/Home/bin/java is an example of Java 11 local path.

Start ResearchSpace

To start the ResearchSpace platform:

  • Download the file ResearchSpace 4.0.0 ZIP.

  • Expand the zip file, open the terminal and move to the downloaded directory with the followig command

    cd yourDirectoryPath/zip-bundle

    NOTE: in the command replace yourDirectoryPath with your local directory path.

  • Start the platform running the followig command from terminal

    For Linux and Mac OS:
                  chmod +x start.sh
              ./start.sh
    For Windows:
    start.bat
  • Open http://localhost:10214/login in your browser.

  • Log into the platform with username:admin and password:admin.

    Log into the platform with username:admin and password:admin

Start ResearchSpace ZIP with example data

To explore ResearchSpace starting with an example of research data
follow these steps:
  • Download and expand the zip file example-runtime-data-for-zip.
  • Open the folder researchspace previously downloaded for the installation with ZIP.
  • Inside the researchspace folder replace the folder runtime-data with the folder example-runtime-data-for-zip and rename it again runtime-data.
  • Finally start the ResearchSpace platform with ZIP as described above.
